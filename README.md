# River Pulse
Need python3 and npm.\
Runs on your http://127.0.0.1:5000/ \
http://127.0.0.1:5000/list - just the database visualized\
http://127.0.0.1:5000/data - the raw database json file\
http://127.0.0.1:5000/dummy - how the source website normally looks (for ease of access/testing) 

To run: \
&ensp; npm run build\
&ensp;&ensp; - builds and runs the server
